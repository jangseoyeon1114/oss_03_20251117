### top, ps, jobs, kill을 중심으로

1. `top`
- 역할
  : 현재 시스템의 *CPU/메모리 사용량, 실행 중인 프로세스, 로드(avg)* 등을 실시간으로 보여 주는 모니터링 도구이다.
  - 화면에서 보이는 주요 정보
    : PID, USER, %CPU, %MEM, TIME+, COMMAND
